# Specification

## Summary
**Goal:** Add an "About Station" section that administrators can manually update through a dedicated admin interface.

**Planned changes:**
- Add "About Station" section to the About page displaying station title and description
- Create backend data structure and storage for station information
- Build admin page with form to edit station title and description
- Implement backend mutation function with admin authorization to update station info
- Add React Query hooks for fetching and updating station information
- Add navigation link from Admin Dashboard to the new Manage Station Info page

**User-visible outcome:** Visitors can view station information on the About page, and administrators can update this content through a new management page in the Admin Dashboard.
